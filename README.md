## 🛠 Popular Tools for Solving Challenges
### Recon & OSINT
- 🌐 Epios OSINT Tool
- 📷 Google Lens

### Web & Application
- 💻 Burp Suite
- 🕸️ wfuzz
- 📂 Gobuster / dirb / dirbuster

### Crypto & Ciphers
- 🔐 CyberChef
- 🧩 DCode (Cipher Identifier)
- ⚡ Aperisolve

### Forensics & Metadata
- 📝 ExifTool
- 🧾 Volatility (memory forensics)
